# MUSTDO API

## Description

Just trying [Nest](https://github.com/nestjs/nest) framework with [Prisma](https://www.prisma.io/).

Just practicing

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
